## Install

[npm][]:

```sh
npm install kepserverex-js
```

[yarn][]:

```sh
yarn add kepserverex-js
```


## Usage

```js
const KepserverexJs = require('kepserverex-js');

const kepserverexJs = new KepserverexJs();

console.log(kepserverexJs.renderName());
// script
```
